# Student Management System (Java)

## Description
This is a console-based Java application developed using Core Java.
It allows users to add, view, and delete student records.
Data is stored using file handling so that records are not lost.

## Concepts Used
- Core Java
- OOP (Class & Object)
- ArrayList
- File Handling
- Exception Handling
